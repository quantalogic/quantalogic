# QuantaLogic Documentation

## Prerequisites

- Python 3.12+
- mkdocs
- mkdocs-material

## Local Development

```bash
# Install dependencies
pip install mkdocs mkdocs-material mkdocs-mermaid2-plugin

# Serve documentation locally
mkdocs serve
```

## Building Documentation

```bash
# Build static site
mkdocs build
```

## Deployment

```bash
# Deploy to GitHub Pages
mkdocs gh-deploy
```
